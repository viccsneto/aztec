Detailed Description

ContactSolver

Solve what to do when collisions occurs. This will dispatch events such as OnBeginContact, OnEndContact to the LuaSubSystem, mark objects to be rendered(culling) when in range of active camera and tracks if the objects are under contact with the mouse cursor.
