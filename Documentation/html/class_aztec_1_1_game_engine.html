void Aztec::GameEngine::beginLoadGameState ( GameStateController *  state)

Puts a GameStateController object in the stack of loading GameStates. This stack will be used to associate new GameObjects within the contextual GameStateController(i.e.: New game object will be associated to the loading GameStateController instead of the active GameStateController)

Parameters
stateA pointer to the GameStateController that will be pushed onto the stack
