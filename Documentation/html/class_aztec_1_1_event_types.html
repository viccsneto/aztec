EventTypes::OnAnimationStarted | EventTypes::OnAnimationFinished | EventTypes::OnBeforeDraw | EventTypes::OnAfterDraw

ALL

The object has all event handlers. Initially(at the first iteration of the gameloop over the object) all GameObjects have this flag setted. After the engine realizes that the GameObject don't have an event handler for any particullar event, it removes the corresponding flag preventing future attempts.
